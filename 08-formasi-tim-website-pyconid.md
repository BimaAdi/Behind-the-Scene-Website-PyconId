# Formasi Tim Website PyCon ID

- Open contributor
    - siapapun bisa jadi open contributor
    - bisa mengambil issue yang ada (kecuali issue khusus core team)
    - bisa datang dan pergi

- Core contributor
    - timnya sudah dipilih
    - bisa mengambil semua issue
        - issue khusus core team (misal: deployment, implementasi payment)
    - wajib mengambil issue jika tidak ada issue yang diambil open contributor
