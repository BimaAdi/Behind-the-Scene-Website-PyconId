# How to Cotribute

## Mengerjakan Task
1. semua task akan dibuatkan issue di github
2. contributor bisa claim issue
3. contributor mengerjakan issue
4. contributor membuat pull request
5. core team akan mereview pull request
6. jika sudah sesuai, pull request akan di merge, jika belum sesuai, core team akan memberikan feedback untuk diperbaiki

## Bug and Feature Request
1. jika menemukan bug atau ingin request fitur, buat issue di github
2. core team akan mereview issue
3. jika issue sesuai, core team akan menambahkan label dan milestone pada issue
4. setelah itu akan mengikuti proses di bagian "Mengerjakan Task"
